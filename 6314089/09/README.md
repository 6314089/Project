#MIDI
midiについて調べて実験したりするブランチ．

とりあえず当面はファイル形式の理解とかをする．

我々のデータをMIDIにあわせるのか，
MIDIデータを我々のデータに合わせて表示するのか，
あるいは別の形式にして見やすく整えるのかは後で決める．
